npm publish to rule them all

  • Build dist files before npm publish.

    ``` { ... "scripts": { "prepublishOnly": "distjs", ... } } ```

  • Files are available on every publish on jsdelivr.

    ```html <script src="https://cdn.jsdelivr.net/npm/pkgname@latest/dist/pkgname.min.js" > </script> ```
